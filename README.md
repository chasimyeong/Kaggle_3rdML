## 대회 요약

### 1. Overview
많은 차종을 구별하는 것은 사람일지라도 쉽지가 않은데요. 만약, 그림만 보고 자동차의 모델(Class)을 쉽게 예측할 수 있다면 CCTV나 블랙박스 속의 차량을 자동으로 분류, 감지해서 범죄예방에 도움을 줄 수 있는 등 다양한 곳에 유익하게 사용될 수 있습니다.
